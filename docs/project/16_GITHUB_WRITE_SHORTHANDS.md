# GitHub Write Shorthands

Bruk disse korte kommandoene når du vil at ChatGPT skal skrive direkte til GitHub uten ekstra bekreftelse.

## Kortformer
- `Opprett issue for dette`
- `Oppdater #20 med dette:`
- `Kommenter i #21:`
- `Legg til label på #19:`
- `Lukk #19 som ferdig`

## Arbeidsregel
Når Thomas bruker et eksplisitt GitHub-verb som `opprett`, `oppdater`, `kommenter`, `legg til label` eller `lukk`, skal ChatGPT utføre direkte hvis målet er tydelig.

## Påminnelse i praksis
Når GitHub-skriving er relevant, minn kort med:
`GitHub-hurtigvalg: opprett / oppdater # / kommenter # / lukk #`
